# Forums

Forums are top-level discussion containers.

## Endpoints

### List Forums

`GET /forums`

### Create Forum

`POST /forums`

Request body:

```json
{
  "title": "Announcements",
  "description": "Official updates"
}
```

### Retrieve Forum

`GET /forums/{forumId}`

### Update Forum

`PUT /forums/{forumId}`

### Delete Forum

`DELETE /forums/{forumId}`
