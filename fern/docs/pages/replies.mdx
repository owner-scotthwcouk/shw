# Replies

Replies belong to posts.

## Endpoints

### List Replies

`GET /posts/{postId}/replies`

### Create Reply

`POST /posts/{postId}/replies`

```json
{
  "content": "Thanks for the update!",
  "author": "Scott"
}
```

### Retrieve Reply

`GET /replies/{replyId}`

### Update Reply

`PUT /replies/{replyId}`

### Delete Reply

`DELETE /replies/{replyId}`
