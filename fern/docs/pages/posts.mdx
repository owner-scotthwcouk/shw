# Posts

Posts belong to topics.

## Endpoints

### List Posts

`GET /topics/{topicId}/posts`

### Create Post

`POST /topics/{topicId}/posts`

```json
{
  "content": "This is a discussion post.",
  "author": "Scott"
}
```

### Retrieve Post

`GET /posts/{postId}`

### Update Post

`PUT /posts/{postId}`

### Delete Post

`DELETE /posts/{postId}`
