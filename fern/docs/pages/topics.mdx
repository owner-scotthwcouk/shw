# Topics

Topics belong to a forum.

## Endpoints

### List Topics

`GET /forums/{forumId}/topics`

### Create Topic

`POST /forums/{forumId}/topics`

```json
{
  "title": "Release Discussion",
  "author": "Scott"
}
```

### Retrieve Topic

`GET /topics/{topicId}`

### Update Topic

`PUT /topics/{topicId}`

### Delete Topic

`DELETE /topics/{topicId}`
